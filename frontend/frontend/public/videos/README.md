Coloca aquí el video que se mostrará en la sección "Biblioteca" del login.

Nombre recomendado de archivo:
- `hand-reading.mp4`

Opcional: poster (imagen estática para carga previa):
- `hand-reading-poster.jpg`

Recomendaciones de encoding:
- Contenedor: MP4
- Codec de video: H.264 (AVC)
- Codec de audio: AAC (o sin audio)
- Ideal: sin audio o audio muy bajo; el elemento se reproduce en `muted` por accesibilidad.
- Resolución recomendada: 1280x720 o 1920x1080, pero mantén bitrate moderado (ej. 2-6 Mbps) para web.

Ruta de uso (desde la app): `/videos/hand-reading.mp4` (coloca el archivo en esta carpeta `public/videos`).

Si quieres que lo descargue yo desde un recurso público, indícame la URL y lo añado (siempre que sea libre de usar).
